<h1><strong>2 задача.</strong></h1>

<h1><strong>2 задача.</strong></h1>

числа.
Всяко положително число наричаме автоморфно, ако се съдържа в края на квадрата си.<br>
<strong>Примери:</strong><br>
1(1), 5(25), 6(36), 25(625)
